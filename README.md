안드로이드 스튜디오와 파이어베이스를 이용해 만든 캘린더 앱입니다.
캘린더 기능과 함께 파이어베이스 데이터베이스에 유저 정보를 저장, 데이터베이스에 각 종 행사가 추가되면 유저가 원하는 행사가 추가되면 자동으로 일정에 추가됩니다.
